This game is a tower defense game, with two modes: freeplay mode and battle mode.

In freeplay mode, you can place polygons anywhere for both sides. The polygons have two parameters: rarity and level. There are nine rarities, ranging from Common to Transcendent, and the level ranges from 1 to infinity for each polygon. There are 8 types of polygons: Shooter, Defender, Attacker, and Exploder for both sides 1 and 2.

There are 4 main settings for cooldown, place on other side, cost, and health. The cooldown setting just enables and disables having a cooldown on each polygon. This defaults to no on freeplay but is forced to yes on battle. The place on other side setting allows or disallows a player to place a polygon on the opposite side. This defaults to yes on freeplay but is forced to no on battle. The cost setting makes or doesn't make placing polygons cost matter. Within the cost settings, there is a set to button and an increase by setting. The set to button sets the current matter to the number in the following text box. The increase by setting increases that side's matter by 1 per that many seconds. Health works similarly, but health is not regained.

You place polygons by clicking, or by using keyboard shortcuts.

List of keyboard shortcuts in freeplay mode:
  • Switch current selected polygon to side 1 shooter: 1
  • Switch current selected polygon to side 1 defender: 2
  • Switch current selected polygon to side 1 attacker: 3
  • Switch current selected polygon to side 1 exploder: 4
  • Switch current selected polygon to side 2 shooter: 7
  • Switch current selected polygon to side 2 defender: 8
  • Switch current selected polygon to side 2 attacker: 9
  • Switch current selected polygon to side 2 exploder: 0
  • Switch current selected polygon rarity to Common: Shift + c (c)
  • Switch current selected polygon rarity to Uncommon: Shift + n (N)
  • Switch current selected polygon rarity to Rare: Shift + r (R)
  • Switch current selected polygon rarity to Epic: Shift + e (E)
  • Switch current selected polygon rarity to Super: Shift + s (S)
  • Switch current selected polygon rarity to Legendary: Shift + l (L)
  • Switch current selected polygon rarity to Mythical: Shift + m (M)
  • Switch current selected polygon rarity to Ultra: Shift + u (U)
  • Switch current selected polygon rarity to Transcendent: Shift + t (T)
  • Decrease current selected polygon level by 1: ArrowDown
  • Increase current selected polygon level by 1: ArrowUp
  • Toggle cooldown: Shift + 1 (!)
  • Toggle place on other side: Shift + 2 (@)
  • Toggle matter: Shift + 3 (#)
  • Toggle health: Shift + 4 ($)
  • Toggle sound: m
  • Switch current selected setting to Matter 1: Shift + 5 (%)
  • Switch current selected setting to Matter 2: Shift + 6 (^)
  • Switch current selected setting to Health 1: Shift + 7 (&)
  • Switch current selected setting to Health 2: Shift + 8 (*)
  • Set current selected setting value to number in textbox: Enter
  • Decrease current selected setting's value input: [
  • Increase current selected setting's value input: ]
  • Decrease current selected setting's increase value (only matter): {
  • Increase current selected setting's increase value (only matter): }
  • Move up: w
  • Move left: a
  • Move down: s
  • Move right: d
  • Place polygon: Space
  • Switch to battle mode: b
